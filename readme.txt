== Installation ==
<p>Manual</p>
1. Upload the `ultimate-social-meta` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. In the blog post edit window fill the meta boxes.
<p>Automatic</p>
1. Goto Dasboard Plugin click 'Add New' search for 'Ultimate Social Meta' .
2. Click Install.
3. Activate Plugin.
